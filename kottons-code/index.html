Cross-universe background lore

The Code is the canonical backbone.

Core rule: everything in Kotton’s Code™ flows from The Code, a magical linguistic-energy system born from living glyphs and rooted in Loyalty Lane. Every app, game, ebook, lesson, quest, collectible, and progression system maps back to this same source.

Origin Story

Long before Kotton was born, tiny glowing glyphs floated through the world and powered communication, creativity, emotion, and connection. The Story Keepers protected those glyphs and organized them into the Five Great Codes.

Kotton’s Destiny

Kotton is a Maltese born with Code Sense: the rare ability to see hidden codes inside words, feelings, and people. Her Code Collar amplifies that spark so she can repair broken emotional codes, unlock Dream portals, strengthen friendship bonds, and restore Legacy pathways.
