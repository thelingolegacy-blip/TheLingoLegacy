# Lingo Legacy OS Web Stack v1

This blueprint is implemented as the deployable static website layer for The Lingo Legacy.

## Core identity layer

Every page inherits:

- Industrial Noir aesthetic.
- Dynamic map background fallback for Philly + Cumberland.
- Ambient sound bed toggle staging.
- Lingo.ai assistant dock.
- Crown identity and seasonal overlays.
- Birthday calendar and rollout timeline lane.
- Unified navigation spine.
- Entity-specific identity skins.
