# The Lingo Legacy HQ Shell Blueprint

Phase 1A establishes the HQ operating shell:

- Next.js + TypeScript application structure
- Vercel deployment-ready package scripts
- Global navigation and mobile command menu
- Industrial Noir design tokens
- Background manager hooks for hero image, particles, ambient motion, sound theme, seasonal overlays, and event overlays
- Sound preference persistence through `localStorage`
- Ask Lingo floating assistant placement
- HQ universe map with connected entity portals
- Legacy timeline framework
- Firebase public configuration layer through `NEXT_PUBLIC_FIREBASE_*` variables

Cloudflare DNS/CDN/security remains an external DNS/security configuration layer in front of the Vercel deployment.
