# KottonsCode Digital Brand System

KottonsCode is the pet legacy studio surface for The Lingo Legacy: Industrial Noir atmosphere, glowing collar magic, emotional wellness stories, Mystery Keys, Loyalty Lane rewards, and a companion app path.

## Core brand system

- Visual language: matte black, deep charcoal, steel texture, cyan/teal collar glow, Peaches mystic purple, premium gold highlights.
- Type direction: bold geometric headlines with clean humanist body copy.
- Symbols: pawprint for actions, collar for loyalty and premium moments, key for unlockables, raven for secrets.
- Promise: Where pets become legends.

## Website structure

1. Home: cinematic hero, decode CTA, explainer, characters, how-it-works, story teasers, Loyalty Lane capture.
2. Episodes / Stories: cover grid, episode hero, summary, emotional theme, Legacy Code takeaway.
3. About / Lore: KottonsCode origin, character bios, Avalon / Legacy / community connection.
4. Resources / Blog: pets, emotional wellness, loyalty, family, community, behind the scenes.
5. Contact / Collaborate: partnerships, press kit, social links.

## Companion app screens

- Onboarding: choose pet avatar, name, traits, loyalty level, glow color.
- Home: daily Legacy Code, episode of the day, XP / loyalty progress ring.
- Episodes: watch or read stories, then unlock Mystery Keys.
- Loyalty Lane: streaks, badges, rewards, premium drops.
- Profile: pet card, saved affirmations, favorite episodes, unlocked keys.

## Landing page funnels

- Pet Parents: Turn your pet's story into a legacy. Capture email and offer the first Legacy Code free.
- Educators / Youth Programs: Story-driven emotional learning for kids and families. Offer guide/PDF and partnership form.
- Premium / Membership: Unlock Loyalty Lane with exclusive episodes, Mystery Keys, early drops, and member rewards.

## Blog content system

Categories:

- Pet Stories and Emotions
- Family and Loyalty
- Behind the Scenes
- Mystery Keys and Clues
- Community Spotlights

Post template:

1. Hero image or episode art
2. Short hook paragraph
3. Story / lesson
4. Legacy Code affirmation section
5. CTA to the episode or signup path

## 30-day social system

Weekly cadence:

- 1 Legacy Code quote card
- 1 Kotton / Jada / Kimba / Peaches character clip
- 1 Mystery Key clue post
- 1 behind-the-scenes production post
- 1 community or pet-of-the-week feature
