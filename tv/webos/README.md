# LG webOS TV package plan

This folder is a starter shell for an LG webOS TV app package.

Before submission you still need:

- LG Developer account access
- Final TV-safe icons: `icon.png` and `largeIcon.png`
- Remote-control navigation testing
- Final privacy policy and support URL
- webOS CLI packaging and device/emulator validation

The current shell is intentionally lightweight and points viewers back to the Vercel-hosted HQ until the full TV app experience is designed.
