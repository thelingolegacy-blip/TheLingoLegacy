Gameplay flow diagrams

Match Loop

Matchmaking Flow

Mode select → skill/rank matching → table reservation → partner assignment → bot fallback if needed → ready check → deal animation.

Hand Flow

Deal → bid → partner bid reveal → trick play → trick sweep → scoring → bags/penalties → next hand or match result.

Reconnection Flow

Disconnect detected → bot protects seat → reconnect token restores hand/table state → player resumes mid-match without forfeiting.
