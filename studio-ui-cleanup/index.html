Separate the layers.

Features stay the same, but each visual layer gets its own role, spacing, depth, and motion priority.

Background world

Atmosphere, lights, particles, and environment live behind content with slow movement and low contrast.

Middle content

Cards, characters, machines, draft arenas, and main features float above the world with clear shadows and glow separation.

Front UI

Buttons, menus, wallets, controls, and navigation sit closest to the user with precise spacing and subtle press feedback.

Remove the boxed-in feeling.

Cut heavy rectangles and let the interface breathe with transparent surfaces, soft shadows, and negative space.

Reduce

Dashboard clutter

  • Heavy borders and thick outlines
  • Buttons touching panels
  • Images competing with copy
  • Too many same-weight rectangles
  • Foreground blending into background
Replace

Floating studio UI

  • Glassmorphism with blur behind panels
  • Soft depth shadows and rim glows
  • Transparent overlays and more empty space
  • Perspective tilt and hover/press motion
  • Light reflections instead of hard outlines

Depth spacing rules.

Every object gets a physical stack: object, shadow, glow, and background separation.

Object

Hero element first

Place the slot machine, player card, Kotton character, or draft arena in the center zone before adding support UI.

Separation

Shadow and glow layer

Use one soft shadow below and one subtle glow behind each major object so it floats without a thick outline.

Spacing

Controls breathe

Primary actions, wallet, menus, and text should sit in separate clusters with consistent gaps instead of one crowded row.

Production-ready mobile layout.

Mobile screens need zones, not a squeezed desktop grid.

Top zone

Logo, profile, currency, and status.

🃏Center zone

Main gameplay or hero object only.

Bottom zone

Navigation and primary actions with thumb spacing.

Mobile rule

Do not place everything in one visual area.

Top status, center hero, and bottom actions should each have their own vertical space. Secondary controls move behind menus, rails, or contextual sheets.

  • Top: logo, profile, currency
  • Center: main gameplay or hero object
  • Bottom: navigation and primary action

Animation hierarchy.

Motion should guide attention, not make every object compete.

Strongest

Main action

Spin, draft pick, card reveal, or character action gets the boldest animation.

Medium

Rewards

Coins, XP, badges, and unlocks get celebratory but shorter motion.

Slow

Background

Lights, particles, fog, and camera drift move slowly and never steal focus.

Subtle

UI controls

Buttons and menus use small hover, press, shimmer, and focus states.
