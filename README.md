# TheLingoLegacy — Website Repo

This repo contains the public website skeleton for the Lingo Legacy™ project.

## Current sections

- `index.html`: root redirect into the public launch page.
- `landing/`: Genesis Launch HQ and Sprint 1–15 Command Center content.
- `assets/`: placeholder logos and images.
- `docs/`: site docs, deployment notes, roadmap briefs, and launch kits.

## Launch package

The Genesis Launch Activation package is available in two formats:

- Web page: `landing/index.html`
- Build brief: `docs/genesis-launch-kit.md`

## Roadmap package

The Sprint 1–15 execution package is available at:

- Build brief: `docs/sprint-roadmap-1-15.md`

## Deployment

This is a static HTML site. No build step is required.

Production notes:

- Connect a commerce checkout provider before accepting payments or pre-orders.
