# TheLingoLegacy — Website Repo

This repo contains the public website skeleton for the Lingo Legacy™ project.

## Current sections

- `landing/`: public landing page and Sprint 1–15 Command Center.
- `assets/`: placeholder logos and images.
- `docs/`: site docs, deployment notes, and roadmap briefs.

## Roadmap package

The Sprint 1–15 execution package is available in two formats:

- Web page: `landing/index.html`
- Build brief: `docs/sprint-roadmap-1-15.md`

## Deployment

This is a static HTML site. No build step is required.
